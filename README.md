# Portfolio Website

The portfolio website can be found at https://prayasmitra.dev

## Installation

### Cloning the project
```bash
git clone https://github.com/notprayasmitra/portfolio-website
cd portfolio-website
```

### Installing packages
```bash
cd frontend/
npm i
cd ../backend/
npm i
```

### Hosting the website locally
```bash
cd frontend/
npm run dev
```
(The frontend should be running on port 3000)

#### In another terminal instance
```bash
cd backend/
npm run dev
```
(The backend server should be running on port 3001)
